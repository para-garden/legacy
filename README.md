# legacy

what humanity left behind.

a worldbuilding project rendered as a spatial graph. factions, figures, events, concepts — positioned because of how they relate, not listed because of when they were written.

not a timeline. not a story with a beginning. a world with facets — coexisting forces, not separate worlds. content carries multiple facets as tags:

hubris, consequences, unsettlement, futility, aspiration, domestication, justice, complacency, choice, satisfaction, rounding.

built on the [ptera.world](https://ptera.world) graph engine.
